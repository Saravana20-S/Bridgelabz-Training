## 📖 Overview

This repository contains programs, assignments, and mini-projects completed during Java training.

The projects cover a wide range of concepts including:

- Java Fundamentals
- Core Java Programming
- Object-Oriented Programming
- Collections Framework
- Data Structures
- File Handling
- JUnit Testing
- Problem Solving

The repository is organized into separate modules, each focusing on a specific concept or use case.
